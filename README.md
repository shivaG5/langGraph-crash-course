# LangGraph Crash Course

A hands-on repository documenting my journey of learning and building applications with LangGraph. This repository covers graph-based workflows, AI agents, memory management, RAG pipelines, and agentic AI concepts through practical implementations.

## Topics Covered

### Graphs

* Graph-I : Hello World Graph
* Graph-II : Multi Input Graph
* Graph-III : Sequential Graph
* Graph-IV : Conditional Graph
* Graph-V : Looping Graph

### Agents

* Agent-01 : Simple Bot
* Agent-02 : Chatbot with Memory
* Agent-03 : ReAct Agent
* Agent-04 : Drafter Agent
* Agent-05 : RAG Agent

## Tech Stack

* Python
* LangGraph
* LangChain
* Ollama
* Google Gemini API
* FAISS
* ChromaDB
* Jupyter Notebook
